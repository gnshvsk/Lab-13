Лабораторная работа №13
Работа с динамическим массивом ArrayList

Выполнила: Гонашевская З.М., гр. ИВТ-173

Задание: с помощью динамического массива ArrayList создать квадратную матрицу.
Подсчитать средние арифметические значения главной и побочной диагоналей матрицы.
Наибольшее значение поместить в середину матрицы вместо исходного числа.

Программа выводит на экран исходную матрицу, средние арифметические значения, наибольшее число и матрицу с наибольшим числом в центре.

Для запуска программы необходимо открыть файл run.bat
